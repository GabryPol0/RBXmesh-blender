# RBXmesh-blender
A simple Blender add-on that allows you to import & export directly as .mesh (2.0) for roblox! It follows the 2.0 file format for mesh files, you'll be missing out on many of the recently released features ( mesh deformation, facial animations, ecc... )

## How to install:
To install the addon, simply open blender and load it in! That's it. nothing else to do.

## What does it do:
After adding the addon, you'll notice 2 options now available in the import and export options:
The first allows you to import a roblox mesh file, and the other allows you to export the model as a mesh file.

## How to use:

### Loading in a .mesh file

1.  Click on import
2.  Select "Roblox Mesh Format"
3.  Select the .mesh file and click import

### Exporting as a .mesh file

1.  Click on export
2.  Select "Roblox Mesh Format"
3.  Give your file a name and export, if you forgot to add .mesh it will automatically add it for you (how sweet)

**P.S: When exporting make sure to export one model at a time!!! The script doesn't automatically iterate through the selected models and only exports the last selected one (sorry :c)**
